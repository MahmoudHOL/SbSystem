# SB Smart License Manager (PyQt)

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
python license_manager.py
```

## Usage

1. Open `http://localhost:3000/license-expired` on target machine.
2. Copy `Machine Fingerprint`.
3. Paste into License Manager.
4. Generate `Trial` or `Permanent`.
5. Save as `license.json`.
6. Put file in:

`<app install folder>/license/license.json`
